PHASE 1: WEB SECURITY BASICS

Learn to identify safe, suspicious, and dangerous websites

‚úÖ SAFE WEBSITES

What makes a website safe?

  • HTTPS: Look for "https://" at the start - the 's' means secure
  • Official domains: hilltophoney.co.uk, royalmail.com, gov.uk
  • Valid certificates: Green padlock icon in browser
  • Correct spelling: No typos in company names
Example: https://www.hilltophoney.co.uk/suppliers

‚ö†Ô∏è SUSPICIOUS WEBSITES

Warning signs to watch for:

  • HTTP only: Missing the 's' - not secure for sensitive data
  • Similar domains: amazon-deals.org instead of amazon.co.uk
  • Unusual extensions: .tk, .ml, .ga domains often used by scammers
  • Extra words: Additional terms added to legitimate domains
Example: http://hilltop-honey-suppliers.net

üö´ DANGEROUS WEBSITES

Never visit these sites:

  • Misspelled domains: hiltophoney.tk (missing 'l')
  • IP addresses: http://192.168.4.23/downloads
  • Fake subdomains: payment-update.company.co.uk.security-check.tk
  • Download sites: Direct links to .exe files from unknown sources
Example: https://hiltophoney.tk/urgent-payment

üéØ Key Points to Remember
